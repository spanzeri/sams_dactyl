# Sam's dactyl manuform

This repository contains both 3d printing sources and firmware for my custom [dactyl manuform](https://github.com/abstracthat/dactyl-manuform).

See the original repo for the model source code.

## 3D printed parts

The initial model was generated using this fantastic [dactyl generator](https://ryanis.cool/dactyl/#manuform).

However, there were few issues with the final model and a couple of tweaks I wanted.

All the models have been heavily modified and re-exported from blender.

## BOM

 * Arduino pro-micro controller (make sure your version dimension match the holder);
 * TRRS for halves connection;
 
For the most part the connection match the diagram in the dactyl repo.

I have moved the TRRS data to D2 because it was convenient to use fewer dupont connector.

## Warning
